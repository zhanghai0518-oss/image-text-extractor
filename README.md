# 图片文字提取工具

从图片中批量提取文字信息，支持自定义字段模板，一键导出Excel。

## ✨ 功能特点

- 📷 **批量导入** - 支持单个文件、多文件、整个文件夹导入
- 📝 **自定义模板** - 可视化配置提取字段，支持字段别名
- 🔍 **智能识别** - 基于Tesseract.js OCR引擎，支持中英文
- 📊 **Excel导出** - 一键导出结构化Excel表格
- ✏️ **结果编辑** - 识别结果可手动修正
- 💾 **模板管理** - 保存和管理多个提取模板

## 🚀 快速开始

### 方式一：直接启动（推荐）

双击 `start.sh` 或在终端运行：
```bash
cd ~/Desktop/image-text-extractor
./start.sh
```

### 方式二：手动启动

```bash
cd ~/Desktop/image-text-extractor
npm install    # 首次运行需要安装依赖
npm start      # 启动应用
```

## 📖 使用说明

### 1. 导入图片

点击左侧「📷 导入图片」或「📂 导入文件夹」按钮，选择要处理的图片文件。

支持的格式：JPG, PNG, BMP, GIF, WebP

### 2. 配置模板

从顶部下拉菜单选择已有模板，或点击「⚙️」创建新模板：

- **字段名**：Excel表格中的列头
- **别名**：图片上显示的字段名（如「纳税人名称」→「所属公司」）
- **启用**：勾选的字段才会被提取

### 3. 提取信息

点击「🔍 提取全部」开始批量识别：

- 进度弹窗显示处理进度
- 可随时取消处理
- 完成后自动显示结果

### 4. 编辑结果

在右侧「提取结果」区域可手动修正识别结果。

### 5. 导出Excel

点击「📊 导出Excel」保存结果表格。

## ⚙️ 高级功能

### 从Excel导入模板

点击「从Excel导入列头」，选择Excel文件，自动读取第一行作为字段名。

### 字段别名说明

别名用于OCR识别时匹配：
- 如果图片上显示「纳税人识别号」但Excel列头是「所属公司代码」
- 设置：字段名=`所属公司代码`，别名=`纳税人识别号`

## 🛠️ 技术栈

- **Electron** - 跨平台桌面应用框架
- **Tesseract.js** - 浏览器端OCR引擎
- **ExcelJS** - Excel文件生成库

## 📝 注意事项

1. 首次启动较慢（需要加载OCR模型）
2. 中英文识别效果最佳，其他语言需额外配置
3. 图片清晰度影响识别准确率
4. 复杂表格可能需要手动调整结果
