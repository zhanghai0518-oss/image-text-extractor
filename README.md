# 图片文字提取工具

[![Release](https://img.shields.io/github/v/release/zhanghai0518-oss/image-text-extractor)](https://github.com/zhanghai0518-oss/image-text-extractor/releases)
[![License](https://img.shields.io/github/license/zhanghai0518-oss/image-text-extractor)](LICENSE)

**专业的税务完税证明数据提取工具** - 一键提取图片/PDF中的税务信息，自动生成结构化Excel表格

---

## ✨ 核心功能

### 📷 智能图片识别
- **Windows**: 集成 PaddleOCR，中文识别率高达 99%
- **macOS**: 使用系统级 Vision OCR，速度快精度高
- **PDF**: 自动提取文本，无需OCR

### 🔄 动态数据提取
- **智能识别税种** - 无需预设列表，自动识别各种税种
- **多记录提取** - 一张凭证多条税目记录自动提取
- **字段自动匹配** - 支持 20+ 字段智能识别

### ✅ 数据校验系统
- **金额验证** - 自动核对记录金额之和与凭证合计
- **字段完整性检查** - 检测关键字段是否为空
- **智能补全** - 金额不匹配时自动尝试补全遗漏记录
- **可视化标记** - 文件列表显示校验状态（✅通过/❗需校验）

### ✏️ 手动纠偏功能（v2.9.21新增）
- **实时编辑** - 在提取结果区域直接修改错误字段
- **翻条浏览** - 支持查看同一文件的多条提取记录
- **保存修改** - 修改后一键保存，同步到数据汇总表格
- **撤销修正** - 可恢复到原始提取结果
- **视觉标记** - 手动修正的记录在表格中高亮显示

### 📊 Excel 批量导出
- **单Sheet导出** - 所有数据合并到一个工作表
- **分Sheet导出** - 按公司/月份自动分组
- **自定义字段** - 支持字段配置导入导出

## 🚀 功能特点

### 1. 高精度OCR识别
```
Windows: PaddleOCR-json (离线、高精度、支持中文)
macOS:   Vision Framework (系统级、速度快)
PDF:     pdf.js 文本提取 (无需OCR)
```

### 2. 智能数据提取
| 特性 | 说明 |
|------|------|
| 动态税种识别 | 自动识别所有税种，无需预设列表 |
| 多格式金额 | 支持逗号分隔、点号分隔、全角￥符号 |
| 日期范围解析 | 自动识别税款所属期起止日期 |
| 备注智能截取 | 截取到税务机关关键字，去除冗余信息 |

### 3. 数据校验机制
- **字段完整性**: 检查所属公司、金额、税种名称等关键字段
- **金额验证**: 记录金额之和 vs 凭证合计金额
- **自动补全**: 发现遗漏记录自动尝试补全
- **状态标记**: 文件列表直观显示校验结果

### 4. 批量处理
- 支持单文件、多文件、整个文件夹导入
- 支持 JPG, PNG, BMP, GIF, WebP 格式
- 支持 PDF 文件（自动提取文本）
- 处理进度实时显示，可随时取消

## 📖 使用说明

### 1. 导入文件
点击「📷 导入图片」或「📂 导入文件夹」，选择税务完税证明图片或PDF

### 2. 配置字段（可选）
- 点击「从Excel导入」读取列头配置
- 或手动配置字段映射

### 3. 提取数据
点击「🔍 提取全部」，等待批量处理完成

### 4. 校验结果
- ✅ 绿色勾：数据完整，金额验证通过
- ❗ 感叹号：有字段为空或金额不匹配，需手动检查

### 5. 手动纠偏（新增）
- 点击需要修正的文件
- 在提取结果区域直接编辑错误字段
- 点击「💾 保存修改」同步到数据汇总表格
- 如需恢复，点击「↩️ 撤销修正」

### 6. 导出Excel
点击「📊 导出Excel」，选择导出模式：
- **单Sheet**: 所有数据合并
- **分Sheet**: 按公司自动分组

## 🎯 字段配置

### 自定义字段
- **灵活配置** - 字段数量不限，可根据实际需求添加任意字段
- **字段映射** - 设置 Excel 列头与识别字段的对应关系
- **别名支持** - 一个字段可设置多个别名匹配
- **启用控制** - 按需启用/禁用字段

### 导入字段配置
- **从 Excel 导入** - 读取 Excel 第一行作为字段名
- **字段配置文件** - 支持导入导出字段配置 JSON

### 常用字段示例
| 字段名 | 图片显示名（别名） |
|--------|-------------------|
| 所属公司 | 纳税人名称、公司名称 |
| 所属公司代码 | 纳税人识别号、统一社会信用代码 |
| 征收机关 | 税务机关 |
| 税票号码 | No、完税证明编号 |
| 缴款日期 | 入库日期、填发日期 |
| 税款所属期起 | - |
| 税款所属期止 | - |
| 税种名称 | 税种 |
| 税目名称 | 品目名称 |
| 金额 | 实缴金额 |
| 备注 | 备注说明 |

> 💡 以上仅为示例，您可以根据实际业务需求自由配置任意数量的字段

## 🔧 技术栈

- **Electron** - 跨平台桌面应用框架
- **PaddleOCR** - 百度开源OCR引擎（Windows）
- **Vision Framework** - Apple OCR框架（macOS）
- **pdf.js** - Mozilla PDF解析库
- **ExcelJS** - Excel文件生成库
